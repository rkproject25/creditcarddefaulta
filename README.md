# 💳 Credit Card Default Prediction (CCDP)

🚀 Live Demo: https://ccd-app.streamlit.app/

---

## 📌 Project Overview

The **Credit Card Default Prediction (CCDP)** project is a machine learning-based web application that predicts whether a credit card user is likely to default on their payment in the next month.

This system helps financial institutions assess customer risk and make better credit decisions using data-driven insights.

---

## 🎯 Objective

The main objective of this project is to:

* Predict credit card default risk using historical financial data
* Analyze customer behavior and payment patterns
* Build an interactive dashboard for real-time prediction
* Compare multiple machine learning models for best performance

---

## 🧠 Goals

* Develop a complete ML pipeline (Preprocessing → Training → Prediction)
* Improve risk assessment accuracy
* Provide business insights for financial decision-making
* Create a user-friendly Streamlit web application
* Enable real-time prediction and scenario simulation

---

## ⚙️ Features

* 🔄 Data Preprocessing (clean & feature engineering)
* 🤖 Multiple ML Models:

  * Random Forest
  * XGBoost

* 📊 Exploratory Data Analysis (EDA)
* 📈 Model Comparison Dashboard
* ⚠️ Default Risk Prediction Tool

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/credit-card-default.git
cd credit-card-default
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run credit_card_default_app.py
```
